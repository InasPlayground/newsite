# Tabs Block - Image-Text Variant

## Usage in Google Docs

To create tabs with image and text content:

```
Tabs

Tab 1 Title
image-text
image1.jpg
This is the text content that appears below the image.

Tab 2 Title
image-text
image2.jpg
More text content here.
```

## Structure

Each tab panel should have:

1. Tab title (first row)
2. Type identifier: "image-text" (second row)
3. Image (third row)
4. Text content (fourth row onwards)

The image will be displayed at the top, with the text content below it.
